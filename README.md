# hex-convert

转换16进制字符串到16进制

## 安装

### 通过 VSCode 插件市场安装

- 打开 VSCode
- 在侧边栏中点击扩展
- 搜索插件名称
- 点击安装

### 手动安装

- 下载插件的 .vsix 文件
- 打开 VSCode
- 在侧边栏中点击扩展
- 点击扩展菜单栏中的 ... 按钮，选择从 VSIX 安装
- 选择下载的 .vsix 文件
